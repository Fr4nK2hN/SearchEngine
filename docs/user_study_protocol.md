# 搜索引擎用户行为研究协议

## 研究目标
调查不同类型的非相关搜索结果对用户行为和搜索效率的影响，通过操控搜索引擎结果页面(SERP)中结果的一致性和相关性，评估用户交互模式，包括搜索时间和查询放弃的可能性。

## 实验设计

### 实验条件
1. **最优模式 (Optimal)**: 高质量、高相关性结果
2. **降级相关性 (Degraded Relevance)**: 混合相关和不相关结果
3. **高度不相关 (Highly Irrelevant)**: 主要为不相关结果
4. **混合一致性 (Mixed Coherence)**: 结果质量不一致

### 实验设计类型
- **被试间设计 (Between-subjects)**: 每个参与者只体验一种实验条件
- **平衡设计**: 每个条件分配相等数量的参与者
- **随机分配**: 参与者随机分配到不同条件

## 参与者要求

### 纳入标准
- 年龄: 18-65岁
- 具备基本计算机和互联网使用技能
- 正常或矫正后正常视力
- 能够理解并同意参与研究

### 排除标准
- 搜索引擎开发或信息检索专业背景
- 严重视觉或认知障碍
- 无法完成60分钟实验会话

### 样本量
- 每个实验条件: 30名参与者
- 总样本量: 120名参与者
- 功效分析: 80%功效检测中等效应量

## 实验流程

### 预实验阶段 (10分钟)
1. **欢迎和介绍** (2分钟)
   - 研究目的简介
   - 实验流程概述
   - 隐私和数据使用说明

2. **知情同意** (3分钟)
   - 详细说明参与权利
   - 签署同意书
   - 确认可以随时退出

3. **背景信息收集** (5分钟)
   - 人口统计学信息
   - 搜索经验问卷
   - 技术熟练度评估

### 训练阶段 (5分钟)
1. **系统介绍** (2分钟)
   - 搜索界面演示
   - 基本操作说明
   - 注意事项提醒

2. **练习任务** (3分钟)
   - 2-3个简单搜索任务
   - 确保参与者熟悉系统
   - 回答操作问题

### 正式实验阶段 (35分钟)
1. **任务执行** (30分钟)
   - 完成25个基线查询中的随机15个
   - 每个查询最多3分钟
   - 自然搜索行为，无时间压力提醒

2. **即时反馈收集** (5分钟)
   - 每5个查询后简短满意度评分
   - 结果相关性感知评估
   - 任务难度评价

### 后实验阶段 (10分钟)
1. **整体体验评估** (5分钟)
   - 系统可用性问卷
   - 整体满意度评分
   - 改进建议收集

2. **访谈和总结** (5分钟)
   - 开放式反馈
   - 实验体验讨论
   - 感谢和结束

## 数据收集标准

### 行为数据 (自动收集)
- **查询数据**: 查询文本、时间戳、会话ID
- **点击数据**: 点击位置、时间、结果ID
- **滚动数据**: 滚动深度、速度、停顿
- **停留时间**: 页面停留、结果查看时间
- **导航数据**: 页面切换、返回行为

### 主观数据 (问卷收集)
- **满意度评分**: 1-5李克特量表
- **相关性感知**: 1-5李克特量表
- **任务完成感**: 是/否 + 置信度
- **认知负荷**: 1-5李克特量表
- **系统可用性**: SUS问卷

### 数据质量控制
- **实时监控**: 异常行为检测
- **数据验证**: 完整性和一致性检查
- **备份机制**: 多重数据存储
- **匿名化**: 个人信息保护

## 参与者指导

### 实验前指导
```
欢迎参与我们的搜索行为研究！

在接下来的实验中，您将使用一个搜索引擎来查找信息。请按照以下指导进行：

1. 自然搜索：请像平时使用搜索引擎一样自然地搜索
2. 仔细阅读：请认真阅读搜索结果，判断是否符合您的需求
3. 诚实反馈：请根据真实感受提供评分和反馈
4. 随时提问：如有任何疑问，请随时询问研究人员

记住：这不是测试您的能力，而是评估搜索系统的表现。
没有标准答案，请按照您的直觉和判断进行操作。
```

### 任务指导模板
```
任务 X: 请搜索关于 "[查询主题]" 的信息

目标：找到相关、有用的信息来回答您的疑问
时间：没有严格时间限制，请按自己的节奏进行
操作：
1. 在搜索框中输入您认为合适的查询词
2. 浏览搜索结果
3. 点击您认为相关的结果
4. 如果需要，可以修改查询词重新搜索
5. 完成后请评价这次搜索的满意度

提醒：请像平时搜索一样自然地操作，不用担心"正确"答案。
```

## 伦理考虑

### 隐私保护
- 数据匿名化处理
- 安全数据存储
- 限制数据访问权限
- 实验结束后数据处理说明

### 知情同意
- 详细说明研究目的和流程
- 明确参与者权利
- 提供退出机制
- 确保理解后签署同意书

### 风险最小化
- 无害的搜索任务
- 避免敏感或个人话题
- 提供心理支持联系方式
- 尊重参与者的舒适度

## 数据分析计划

### 描述性分析
- 参与者特征统计
- 行为指标分布
- 实验条件平衡性检查

### 推断性分析
- 条件间差异检验 (ANOVA)
- 效应量计算
- 置信区间估计
- 多重比较校正

### 探索性分析
- 用户类型聚类分析
- 行为模式识别
- 预测模型构建

## 预期结果和影响

### 预期发现
- 不相关结果显著影响用户满意度
- 混合质量结果增加认知负荷
- 用户会采用不同策略应对低质量结果

### 实际应用
- 搜索引擎质量评估标准
- 用户体验优化指导
- 个性化搜索策略开发

### 学术贡献
- 用户搜索行为理论扩展
- 信息检索评估方法改进
- 人机交互设计原则完善
