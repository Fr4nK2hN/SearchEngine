# 搜索引擎系统用户反馈问卷

## 一、说明

您好！

本问卷用于收集用户对本搜索引擎系统的使用反馈，结果仅用于毕业设计研究与系统优化，不涉及商业用途。  
问卷采用 **Likert（李克特）5点量表**，请您根据实际使用体验进行评价。

评分标准如下：

| 分值 | 含义 |
| --- | --- |
| 1 | 非常不同意 |
| 2 | 不同意 |
| 3 | 一般 |
| 4 | 同意 |
| 5 | 非常同意 |
